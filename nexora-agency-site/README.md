# Agency Website

A simple static website for a web design, SEO, website security, and maintenance agency.

## Files

- `index.html` — website content
- `style.css` — design and responsive layout
- `script.js` — mobile menu and demo contact form

## Deploy to Cloudflare Pages

1. Create a GitHub repository.
2. Upload these files to the repository.
3. Go to Cloudflare Dashboard > Workers & Pages > Create application > Pages.
4. Connect your GitHub repository.
5. Use these settings:
   - Framework preset: None
   - Build command: leave blank
   - Build output directory: `/`
6. Click Deploy.

## Customize

Search and replace:

- `NexoraDigital` with your agency name
- `Your Agency Name` with your real business name
- `hello@youragency.com` with your email address
- Pricing, services, and text as needed
